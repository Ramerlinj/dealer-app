# ReySoft Multiservices Dealer App

This project is a [Next.js](https://nextjs.org) application designed to empower dealer networks in the Dominican Republic with digital experiences, actionable data, and local narratives. It is bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

You can start modifying the main page by editing `app/page.tsx`. The page will automatically update as you make changes.

This project utilizes [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to optimize and load [Geist](https://vercel.com/font), a font family by Vercel.

## Key Features

*   **Digital Ecosystems:** Provides digital solutions tailored for Dominican dealers.
*   **Data-Driven Insights:** Offers actionable data to improve dealer performance.
*   **Local Focus:** Emphasizes local narratives and market understanding.
*   **Modern UI:** Built with Next.js and Tailwind CSS for a fast and responsive user experience.
